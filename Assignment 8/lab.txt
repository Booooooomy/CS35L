compute a cosine and a square root, we import the math.h library. Additionally, we import the
stdio.h library to print the answer to 17 significant digits. The program is reproduced below.

//trivial C program
#include <math.h>
#include <stdio.h>

int main (void) {
  printf("%.17g", cos(sqrt(3.0)));
  return 0;
}

Command: gcc -o trivialprog trivialprog.c

and 'fstat' is used to get information about the library, such as the size so it can allocate an

protect the mapping of the library from attacks. Lastly, the 'close' call closes the library once
the code has been dynamically linked.
